### 🚀 **Quick Start**

1. **Setup Database** (you'll need PostgreSQL):
   ```bash
   # Windows
   setup-backend.bat
   
   # Mac/Linux
   chmod +x setup-backend.sh
   ./setup-backend.sh
   ```

2. **Environment Variables** - Create `.env`:
   ```env
   DATABASE_URL="postgresql://user:password@localhost:5432/modernmen"
   NEXT_PUBLIC_API_URL="http://localhost:3000"
   ```

3. **Start Development**:
   ```bash
   npm run dev
   ```

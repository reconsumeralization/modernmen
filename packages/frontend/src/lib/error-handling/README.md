## üö® Common Error Patterns

### Array Bounds Errors
```typescript
// ‚ùå Unsafe access
const item = items[index] // May throw if index out of bounds

// ‚úÖ Safe access
const item = safeGet(items, index, null)
if (!item) {
  console.log('Index out of bounds or array empty')
}
```

### Null Reference Errors
```typescript
// ‚ùå Unsafe property access

// ‚úÖ Safe property access
const name = safeProp(user, 'profile.name', 'Unknown')
```

### API Error Handling
```typescript
// ‚ùå Generic error handling
try {
  await apiCall()
} catch (error) {
  console.error('API Error:', error)
}

// ‚úÖ Structured error handling
try {
  await apiCall()
} catch (error) {
  if (error instanceof EnhancedAPIError) {
    // Handle specific API errors
    if (error.code === ErrorCode.UNAUTHORIZED) {
      redirect('/login')
    }
  }
  await captureError(error, { component: 'APIHandler' })
}
```
